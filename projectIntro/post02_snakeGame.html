Snake Game 프로젝트

Snake Game 설명

Posted by 최재호 | 2021 Aug 25

프로젝트 기간 : 진행중(약 2주)
참여인원 : 풀스택 1명
담당 : 풀스택 및 배포
프로젝트 소개 : 사용자가 웹상에서 뱀게임을 플레이 하며 점수를 기록할수 있습니다. 사용자는 화살표를 이용하여 뱀을 움직일수 있습니다. 먹이(노란색)를 먹을때마다 길이가 길어지며 벽이나 몸통에 부딪혔을때는 게임오버가 됩니다. 간단하게 id와 password를 입력하고 snake game을 진행할수 있으며 최고 점수는 DB에 저장되고 순위도 확인할수 있습니다.
현재 게임부분은 완료가 됬고 서버부분을 개발중에 있습니다.
